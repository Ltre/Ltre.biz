# 申报价模板管理工具

一键将申报价模板批量套用到按国家区分的申报价表格中，避免运营人员每次手动逐个填写。

## 快速使用

1. 打开目标网页的申报价弹窗（确保表格已渲染）
2. `F12` 打开浏览器控制台（Console）
3. 粘贴 `country-price-template.js` 全部内容，回车
4. 弹出管理面板即可操作

## 功能

| 操作 | 说明 |
|------|------|
| **+ 新增模板** | 输入模板名后进入二维表格编辑，当前表格有哪些国家就列出哪些国家 |
| **📋 从当前表格创建** | 一键抓取当前页面上所有国家及其已填的申报价，保存为模板 |
| **编辑** | 修改已有模板的国家和价格，编辑时自动同步最新国家列表 |
| **套用** | 将模板中每个国家的申报价批量填入当前表格 |
| **删除** | 删除不需要的模板 |

## 匹配逻辑

- 套用模板时按 `trim()` 后的国家名逐行遍历匹配，不依赖顺序
- 即使 UI 上国家名有空格、换行等脏字符，也能正确找到对应行
- 模板中存了但表格中没有的国家会被跳过，不会报错

## 存储

模板保存在浏览器 `localStorage` 中，关闭浏览器后不会丢失。不同域名之间隔离。

## 注意事项

- 页面表格的 DOM 结构变化后需调整选择器（文件顶部 `ROOT_SEL`）
- 使用了 Vue 的原生 value setter + 事件派发，确保 Vue 组件感知到值变化
